# LLM API Setup (LBL)

Some notes below explain how to obtain API keys for each platform.

Save the following API key environment variables to `~/.bashrc`:

```bash
export OPENAI_API_KEY=<...>
export GEMINI_API_KEY=<...>
export CBORG_API_KEY=<...>
```

## OpenAI

Sign up for the LBL ChatGPT Team using the Google Form.

Enter Project/Activity ID: 110260 001

Sign in here to view API keys: https://platform.openai.com/settings/organization/api-keys

Click on "API Keys" under the Project tab to create an API key

## Gemini

Go to: https://aistudio.google.com/

Click on "Get API Key"

Managing project members can be done from the Google Cloud console: https://console.cloud.google.com/iam-admin/iam
